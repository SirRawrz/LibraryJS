Stream Archiver Options

Point this extension at the page or root URL that contains platform.txt and serverip.txt. The extension resolves the upload host from serverip.txt and writes into /videodownloader/.
Used to rank which subtitle preview looks most like English. You can swap these words for another language later.
Turn this off to skip subtitles entirely during archive uploads. Text downloads stay off unless you enable them, so HTML/TXT files will not clutter the list. The GIF/TXT ignore switch also suppresses plain .gif and .txt captures even if text capture is enabled. Set segment delay to 0 to disable the old throttle. Reserve buffer overestimation controls the extra percent added to the packet budget guess used for reserve sizing. Leave Mediabunny bundle URL blank to use the bundled mediabunny/ folder.
Platform values are read from platform.txt and default to windows. The server IP/port is read from serverip.txt and used to upload into /videodownloader/. The archiver now ignores its own saved output so it does not loop on uploaded playlists.
